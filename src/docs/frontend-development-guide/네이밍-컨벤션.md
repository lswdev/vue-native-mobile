## 1. 파일 네이밍

| 대상 | 규칙 | 예시 |
|------|------|------|
| Vue 컴포넌트 | PascalCase | `BaseButton.vue`, `GisMapPanel.vue` |
| 페이지 컴포넌트 | PascalCase + `Page` suffix | `HomePage.vue`, `LoginPage.vue` |
| 레이아웃 컴포넌트 | PascalCase + `Layout` suffix | `DefaultLayout.vue` |
| Composable | camelCase + `use` prefix | `useAuth.ts`, `useGisMap.ts` |
| Pinia 스토어 | camelCase + `.store` | `auth.store.ts` |
| 서비스 | camelCase + `.service` | `energy.service.ts` |
| 유틸리티 | camelCase + `.util` | `date.util.ts` |
| 타입 파일 | camelCase + `.types` | `api.types.ts` |
| CSS Module | camelCase + `.module.css` | `gisMapPanel.module.css` |
| CSS 클래스 (BEM) | kebab-case · BEM | `gis-map-panel__header--active` |

## 2. CSS 클래스 네이밍 (BEM)

커스텀 CSS 클래스(`<style scoped>`, CSS Modules, `@layer components`)는 **BEM(Block Element Modifier)** 을 따른다.

| 구분 | 패턴 | 예시 |
|------|------|------|
| Block | `{block}` | `gis-map-panel` |
| Element | `{block}__{element}` | `gis-map-panel__legend` |
| Modifier | `{block}--{modifier}` 또는 `{block}__{element}--{modifier}` | `gis-map-panel--collapsed`, `gis-map-panel__legend--hidden` |

- Block 이름은 **kebab-case**만 사용한다.
- 상세 규칙·Tailwind 병행·Vue SFC 적용 예시는 [스타일링 가이드 BEM 네이밍 규칙](./스타일링-가이드.md#bem-네이밍-규칙)을 참고한다.

## 3. 변수/함수 네이밍

```typescript
// ✅ 올바른 예
const userName = 'admin'                  // camelCase
const MAX_RETRY_COUNT = 3                 // SCREAMING_SNAKE_CASE (상수)
interface UserProfile { ... }             // PascalCase (인터페이스)
type ApiResponse<T> = { ... }             // PascalCase (타입)
enum UserRole { Admin = 'ADMIN' }         // PascalCase (enum)

// 함수
function fetchUserData() { ... }          // camelCase 동사형
const handleSubmit = () => { ... }        // handle + 이벤트명

// Pinia 스토어 ID
defineStore('auth', ...)                  // kebab-case 또는 camelCase
defineStore('gis-map', ...)

// ❌ 잘못된 예
const UserName = 'admin'                  // 변수에 PascalCase 금지
const fetch_user = () => { ... }          // snake_case 금지
```

## 4. 컴포넌트 이벤트 네이밍

```typescript
// emit 이벤트: kebab-case
const emit = defineEmits<{
  'update:modelValue': [value: string]
  'item-click': [item: Item]
  'data-loaded': [data: DataType[]]
}>()
```
